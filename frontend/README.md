### URLs da Aplicação

As URLs são adaptadas ao contexto do sistema, mantendo termos técnicos já consolidados no mercado e utilizando português em áreas relacionadas ao domínio do negócio.

Exemplos:

```
/login
/dashboard
/empresas
/clientes
```

Exemplo:

```
http://localhost:5173/empresas
```

utiliza internamente:

```
CompaniesPage.tsx
```

A camada técnica permanece em inglês, enquanto a interface do usuário utiliza português brasileiro.
